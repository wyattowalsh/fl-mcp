## Purpose

Provide deterministic diagnostics and remediation steps for common FL MCP issues.

## Prerequisites

- Reproduction steps for the failing behavior.
- Access to CLI diagnostics output and docs check/build logs.
- Current branch state with latest generated references.

## Steps

1. Reproduce the issue with minimal context.
2. Run diagnostics and capture exact errors.
3. If docs dependencies are involved, reinstall from the docs-local lock with
   `pnpm --dir docs --ignore-workspace install --frozen-lockfile`.
4. Validate documentation health with `pnpm --dir docs --ignore-workspace check`.
5. Validate app compilation with `pnpm --dir docs --ignore-workspace build`.
6. Apply targeted fix and rerun failing checks.

## Common Issues

### pnpm Uses A Parent Workspace

If pnpm reports dependencies or build approvals that do not belong to this repo,
rerun the command with `--ignore-workspace`:

```bash
pnpm --dir docs --ignore-workspace install --frozen-lockfile
pnpm --dir docs --ignore-workspace check
```

The docs package is nested under `docs/`; root-level npm files are intentionally
absent.

### `.source` Is Missing

`docs/.source` is generated by Fumadocs and is ignored. Run:

```bash
pnpm --dir docs --ignore-workspace lint
```

The lint script regenerates `.source` before TypeScript checks.

### Generated Reference Drift

If `docs/scripts/check.mjs` reports generated reference drift, regenerate:

```bash
pnpm --dir docs --ignore-workspace docs:generate-reference
```

### Live Bridge Timeout

If live mode times out, capture:

- `FL_MCP_BRIDGE_MODE`
- `FL_MCP_FL_STUDIO_BRIDGE_CMD`
- `FL_MCP_FL_STUDIO_BRIDGE_DIR`
- controller script path
- request id or operation id
- whether the bridge directory has request/response files

An empty bridge directory after timeout usually means the FL Studio MIDI script
is not selected or not polling.

## Verification

- Original failure no longer reproduces.
- Diagnostics no longer report the prior error state.
- Docs lint/check/build are all green.
- Generated build artifacts have been removed or are ignored after validation.
