## Ownership

`docs/scripts/generate-reference.mjs` writes the generated MDX files. Do not edit
files under `docs/content/docs/reference/generated/` by hand.

The public API inventory is hand-maintained because it explains the supported
contract, migration guidance, and governance rules. Keep it aligned with tests,
FastMCP inspection, and generated schemas whenever tool names, resource URIs,
prompt names, operation ids, provider metadata, or output models change.

Run this check before claiming reference docs are current:

```bash
pnpm --dir docs --ignore-workspace docs:verify-reference
```
