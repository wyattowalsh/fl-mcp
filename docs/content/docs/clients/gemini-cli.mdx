## Prerequisites

- Local FL MCP runtime is installed and reachable.
- Gemini CLI profile can load MCP transport/auth settings.

## Configuration

1. Register FL MCP endpoint details in the active Gemini CLI profile.
2. Confirm credentials resolve from expected environment sources.
3. Reload profile/session before validating tool calls.

## Verification

- Gemini CLI can discover FL MCP surface without transport errors.
- Representative commands execute with expected responses.
