## Prerequisites

- Local FL MCP runtime is installed and reachable.
- Claude Code is configured to load local MCP server config files.

## Configuration

1. Point Claude Code to the FL MCP server configuration.
2. Ensure provider/auth settings are available in the active environment.
3. Reload client configuration if required.

## Verification

- Claude Code lists the expected FL MCP tools/resources.
- A representative read + write flow succeeds.
