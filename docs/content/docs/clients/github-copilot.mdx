## Prerequisites

- Local FL MCP runtime is installed and reachable.
- GitHub Copilot MCP integration path is enabled in your environment.

## Configuration

1. Register FL MCP endpoint and transport settings in Copilot integration config.
2. Confirm auth/provider values are available at runtime.
3. Refresh integration state so new endpoints are loaded.

## Verification

- Copilot integration can discover FL MCP endpoints.
- Representative tool calls complete successfully.
