## Purpose

Get a clean first run of FL MCP with validated connectivity.

## Prerequisites

- Node and `pnpm` installed for docs/client tooling.
- Python runtime and project dependencies installed.
- Access to at least one MCP-capable client.

## Steps

1. Install project dependencies.
2. Configure your target provider and client integration.
3. Run initial health/handshake checks.
4. Confirm reference docs are generated and current.

## Verification

- Runtime health endpoint reports healthy status.
- Client can discover expected MCP surface.
- No docs check or build failures.
